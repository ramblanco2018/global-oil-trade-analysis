# Global Oil Trade Analysis (1995–2021)

Structural analysis of global crude oil export dynamics, regional concentration, sanctions impact, and divergence between formal trade statistics and informal flows.

## 📌 Project Overview

This project analyzes global crude oil export data from 1995 to 2021 using UN Comtrade official trade statistics..

The objective is to:

- Identify structural leaders in global oil exports
- Measure regional concentration (Middle East vs Latin America)
- Evaluate the impact of sanctions on formal export flows
- Explore the divergence between reported trade and real market activity

---

## 📊 Key Findings

- Middle Eastern exporters consolidated structural dominance over the 1995–2021 period.
- Latin America's participation declined significantly due to Venezuela's collapse in formal trade.
- Sanctions on Iran and Venezuela correlate negatively with declared exports.
- Formal trade statistics may underestimate real flows due to informal “dark fleet” mechanisms.

---

## 🧠 Methodology

- Data source: UN Comtrade (HS Code 2709 – Crude Petroleum Oils)
- Period analyzed: 1995–2021
- Currency normalization to billions USD
- Regional grouping for structural comparison
- Sanction dummy variable and correlation analysis
- Visualization via Python and interactive HTML dashboards

## ⚠️ Note on Data Interpretation

This project analyzes formal trade declarations.  
Sanctioned countries may continue exporting crude through non-declared or indirect channels, which are not fully captured in official trade statistics.
